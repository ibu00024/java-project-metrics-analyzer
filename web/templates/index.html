使い方

  • このツールはJavaプロジェクトのメトリクスの解析ができます．
    1. 解析したいGitHubリポジトリのURLを入力
    2. Analyzeボタンを押す
  • 現在取得できるメトリクス一覧
メトリクス名 説明
Files リポジトリ内のJavaファイル数
Blank Lines ソースコード内の空白行数
Comment Lines ソースコード内のコメント行数
Code Lines ソースコード内のコード行数
Classes クラスの定義数
Method Declarations メソッドの宣言数
Method Invocations メソッド呼び出し数
Javadocs Of Method メソッドに関連付けられたJavadocの数
All Javadocs リポジトリ内の全てのJavadocの数
